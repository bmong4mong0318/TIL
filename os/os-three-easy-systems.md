## 세그멘테이션

## 18. 페이징: 개요

프로세스의 주소 공간을 몇 개의 가변 논리 세그먼트로 나누는 것이 아니라 고정 크기의 단위로 나눕니다. 이 각각의 고정 크기의 단위를 **페이지(page)** 라고 부릅니다.

### 페이지 테이블

- 주소 공간의 각 가상 페이지에 대한 물리 메모리 위치 기록을 위하여 운영체제는 프로세스 마다
  **페이지 테이블(page table)** 이라는 자료구조를 유지합니다.

- 페이지 테이블의 주요역할: 가상 페이지 **주소 변환(address translation)** 정보를 저장합니다. 각 페이지가 저장된 물리 메모리의 위치를 알려줍니다.

- 페이지 테이블은 프로세스마다 존재합니다.

### 예시

```
movl <virtual address>, %eax
```
